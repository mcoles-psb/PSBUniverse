# Home Application

Route: /

## Purpose

Provides the main navigation dashboard for operational modules and setup pages.

## Main Features

- Categorized tile navigation:
  - Operation
  - Setup
  - Profile
  - Inquiry
- Quick links to:
  - Gutter module
  - OHD module
  - Global setup
  - Gutter setup
  - Company profile
  - Travel module

## Implementation Notes

- Built as a client component.
- Uses React-Bootstrap cards and grid layout.
- Reads no live database data.
- Inherits global compact-density typography/spacing defaults.
